# python-web-flask-mvc-pandas-simple-pie-chart-image

## Description
A POC for showing pandas
pie chart in a flask mvc.

Pie chart ledgend on right
side.

## Tech stack
- python
  - flask
  - matplotlib
  - pandas

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`
- Available http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
